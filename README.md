Human Resources Management System (Java Streams Edition)

Project Overview:
An advanced HR Management System developed in Java that demonstrates object-oriented design. The system manages complex relationships between different entities like Employees and Interns (Students).

Technical Features:
Java Streams: Implements functional programming using `java.util.stream` to filter, sort, and process data efficiently (e.g., filtering employees by salary).

Technologies Used
Language: Java 
Concepts: Functional Programming (Streams), OOP (Inheritance, Polymorphism, Encapsulation), and Method Overriding.
IDE: Eclipse
